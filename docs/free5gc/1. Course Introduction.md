# 1. Course Introduction

## Overview

free5GC is an open source implementation of the 5G Core, aligned with 3GPP Release 15 and designed as a learning and research platform for modern 5G networks. It provides a clear, modular view of how core network functions work together and how service-based architecture is applied in practice.

This course introduces the foundations of free5GC: its overall architecture, the roles of key network functions such as AMF, SMF, and UPF, and the development patterns used throughout the project. You’ll learn how the system is structured, how components interact, and how to navigate the codebase with confidence.

We’ll walk through core concepts, examine how free5GC implements 5G control and user plane behavior, and explore the tools and patterns developers use to run, debug, and extend the system. You’ll develop both conceptual understanding and practical insight, preparing you to build deeper 5G Core skills and begin contributing when you’re ready.

---

## Learning Objectives

By the end of this course, you should be able to:

* Describe free5GC as an open source 5G core network project, including its goals in enabling research, education, and innovation in 5G.
* Explain the architecture of a 5G core network and explain how free5GC implements it using service-based design.
* Identify and summarize the roles of the major 5G core functions (AMF, SMF, UPF, AUSF, etc.) as they are implemented in free5GC.
* Navigate the free5GC codebase to understand how the Go-based implementation and optional GTP5G data plane work together.
* Apply development best practices to build, test, and deploy free5GC in Linux or containerized environments.
* Use debugging and observability tools to analyze free5GC’s behavior and troubleshoot issues.
* Identify ways to get involved with the free5GC project and contribute to its ongoing development.

## Additional Details

### Who is this course for?

This course is designed for learners with a basic understanding of 5G concepts who are ready to take the next step into open source 5G core development. It is ideal for network engineers, developers, and open source enthusiasts interested in understanding how free5GC is structured and how they can begin contributing to the project.

### What does the course prepare you for?

This course will enable learners to understand and navigate the free5GC codebase, develop and modify core network functions, and contribute effectively to the open source 5G community. It builds the foundation needed to advance as free5GC developers who can extend functionality, troubleshoot issues, and collaborate confidently within the project ecosystem.

### What prerequisites should you have?

This course assumes a basic familiarity with Linux and networking concepts. An understanding of 5G core network fundamentals, such as 5GC services and network functions, will be helpful. Programming or scripting experience in Go or shell can make the examples easier to follow, but it is not required. No prior experience with free5GC development is expected, though learners who have previously run free5GC or UERANSIM may find the material more relatable.

### What are the lab environment / system requirements?

* CPU with AVX support
* 4GB or more of RAM is recommended
* A Linux kernel is required
* Ubuntu 20.04^ is recommended
* For other Linux distributions, please ensure the kernel version is greater than 5.0.0.
