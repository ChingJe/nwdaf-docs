<tr class="odd">
<td>start</td>
<td>string</td>
<td>O</td>
<td>0..1</td>
<td><p>First value identifying the start of a TAC range, to be used when the range of TAC's can be represented as a hexadecimal range (e.g., TAC ranges). 3-octet string identifying a tracking area code, each character in the string shall take a value of "0" to "9" or "A" to "F" and shall represent 4 bits. The most significant character representing the 4 most significant bits of the TAC shall appear first in the string, and the character representing the 4 least significant bit of the TAC shall appear last in the string.</p>
<p>Pattern: "^([A-Fa-f0-9]{4}|[A-Fa-f0-9]{6})$"</p></td>
</tr>
<tr class="even">
<td>end</td>
<td>string</td>
<td>O</td>
<td>0..1</td>
<td><p>Last value identifying the end of a TAC range, to be used when the range of TAC's can be represented as a hexadecimal range (e.g. TAC ranges). 3-octet string identifying a tracking area code, each character in the string shall take a value of "0" to "9" or "A" to "F" and shall represent 4 bits. The most significant character representing the 4 most significant bits of the TAC shall appear first in the string, and the character representing the 4 least significant bit of the TAC shall appear last in the string.</p>
<p>Pattern: "^([A-Fa-f0-9]{4}|[A-Fa-f0-9]{6})$"</p></td>
</tr>
<tr class="odd">
<td>pattern</td>
<td>string</td>
<td>O</td>
<td>0..1</td>
<td>Pattern (regular expression according to the ECMA-262 dialect [8]) representing the set of TAC's belonging to this range. A TAC value is considered part of the range if and only if the TAC string fully matches the regular expression.</td>
</tr>
<tr class="even">
<td colspan="5">NOTE: Either the start and end attributes, or the pattern attribute, shall be present.</td>
</tr>

EXAMPLE 1: TAC range. From: 543000 To: 5433E7 (i.e., 1000 TAC numbers)  
JSON: { "start": "543000", "end": "5433E7" }

EXAMPLE 2: TAC range. From: 54E000 To: 54EFFF (i.e., 4096 TAC numbers)  
JSON: { "pattern": "^54E\[0-9a-fA-F\]{3}\$" }, or  
JSON: { "start": "54E000", "end": "54EFFF" }
