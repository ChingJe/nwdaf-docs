<td>roamingUeInd</td>
<td>boolean</td>
<td>O</td>
<td>0..1</td>
<td><p>When present, this IE shall indicate whether the SMSF can serve roaming UE:</p>
<p>- true: the SMSF can support roaming UEs.</p>
<p>- false: the SMSF can not support roaming UEs.</p>
<p>Absence of this IE indicates whether the SMSF can serve roaming UEs is not specified.</p></td>
</tr>
<tr class="even">
<td>remotePlmnRangeList</td>
<td>array(PlmnRange)</td>
<td>O</td>
<td>1..N</td>
<td><p>This IE maybe present when the roamingUeInd IE is present with the value "true".</p>
<p>When present, this IE shall indicate the list of ranges of remote PLMNs served by the SMSF, i.e. the SMSF can serve the roaming UEs which belong to the indicated remote PLMNs.</p></td>
</tr>
<tr class="odd">
<td colspan="5">NOTE: If the roamingUeInd IE is present with the value "true", absence of remotePlmnRangeList IE indicates that the SMSF can serve roaming UEs from any remote PLMN.</td>
