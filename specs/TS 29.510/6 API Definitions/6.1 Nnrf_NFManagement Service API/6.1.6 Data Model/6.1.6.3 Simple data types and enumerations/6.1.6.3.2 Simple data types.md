# 6.1.6.3.2 Simple data types


The simple data types defined in table 6.1.6.3.2-1 shall be supported.

Table 6.1.6.3.2-1: Simple data types

<table>
<colgroup>
<col style="width: 20%" />
<col style="width: 20%" />
<col style="width: 59%" />
</colgroup>
<tbody>
<tr class="odd">
<td>Type Name</td>
<td>Type Definition</td>
<td>Description</td>
</tr>
<tr class="even">
<td>NefId</td>
<td>string</td>
<td><p>The NEF ID as specified in clause 4.25.2 of 3GPP TS 23.502 [3].</p>
<p>For combined SCEF+NEF, the NEF ID shall contain the SCEF ID encoded as specified in clause 8.4.5 of 3GPP TS 29.336 [37].</p></td>
</tr>
<tr class="odd">
<td>VendorId</td>
<td>string</td>
<td><p>Vendor ID, according to the IANA-assigned "SMI Network Management Private Enterprise Codes" [38].</p>
<p>It shall be formatted as a fixed 6-digit string, padding with leading digits "0" to complete a 6-digit length.</p>
<p>Pattern: "^[0-9]{6}$"</p></td>
</tr>
<tr class="even">
<td>WildcardDnai</td>
<td>string</td>
<td><p>String representing the Wildcard DNAI.</p>
<p>It shall contain the string "*".</p>
<p>Pattern: '^[*]$'</p></td>
</tr>
<tr class="odd">
<td>ImsDomainName</td>
<td>string</td>
<td>The IMS domain name.</td>
</tr>
<tr class="even">
<td>MediaCapability</td>
<td>string</td>
<td><p>String representing the media capability offered by NF instance.</p>
<p>Pattern: '^[a-zA-Z0-9_]+$'</p></td>
</tr>
</tbody>
</table>
