# 7.2.28 MB-SMF Services

The following NF service is specified for MB-SMF.

Table 7.2.28-1: NF Services provided by MB-SMF

| Service Name      | Description                                                                           | Reference in TS 23.247 \[129\] |
|-------------------|---------------------------------------------------------------------------------------|--------------------------------|
| Nmbsmf_TMGI       | This service enables NF service consumer to allocate, refresh and deallocate TMGI(s). | 9.1.2                          |
| Nmbsmf_MBSSession | This service enables the consumer to create, update and delete the MBS Session.       | 9.1.3                          |
