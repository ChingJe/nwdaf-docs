# 7.2.27 NSACF Services

The following NF services are specified for NSACF:

Table 7.2.27-1: NF Services provided by NSACF

| Service Name              | Description                                                                                                                                                                                                                                                                                                                                | Reference in TS 23.502 \[3\] |
|---------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------|
| Nnsacf_NSAC               | This service enables consumer NF to check the availability per network slice and update the number of UEs registered with a network slice, or the number of UEs with at least one PDU Session/PDN Connection established on a network slice in the case of EPC interworking, or the number of PDU Sessions established on a network slice. | 5.2.21.2                     |
| Nnsacf_SliceEventExposure | This service enables consumer NF(s) to subscribe and get notified of the event as described in clause 5.15.11.4.                                                                                                                                                                                                                           | 5.2.21.4                     |
