# 7.2.15 BSF Services

The following NF services are specified for BSF as described in TS 23.503 \[45\]:

Table 7.2.15-1: NF Services provided by BSF

| Service Name                                                                                                                                                                                                                               | Description                                                                                         | Reference in TS 23.502 \[3\] |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|------------------------------|
| Nbsf_Management                                                                                                                                                                                                                            | Allows a PCF to register/deregister itself and to be discoverable by NF service consumers (NOTE 1). | 5.2.13.2                     |
| NOTE 1: There may be both, a PCF for the PDU Session and a PCF for the UE. Each of them may separately and independently register itself at the BSF. Each of them may separately and independently be discovered by a consumer of the BSF. |                                                                                                     |                              |
