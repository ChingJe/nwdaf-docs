# 5.3.6 Data Model


## 5.3.6.1 General


This clause specifies the application data model supported by the API.

Table 5.3.6.1-1 specifies the data types defined for the Nnwdaf_DataManagement service based interface protocol.

Table 5.3.6.1-1: Nnwdaf_DataManagement specific Data Types

|                           |                |                                                                                                            |               |
|---------------------------|----------------|------------------------------------------------------------------------------------------------------------|---------------|
| Data type                 | Clause defined | Description                                                                                                | Applicability |
| NnwdafDataManagementSubsc | 5.3.6.2.2      | Represents an Individual NWDAF Data Management Subscription resource.                                      |               |
| NnwdafDataManagementNotif | 5.3.6.2.3      | Represents a notification that corresponds with an Individual NWDAF Data Management Subscription resource. |               |
| PendingNotificationCause  | 5.3.6.3.3      | Represents the Pending Notification Cause for the stored unsent events.                                    | EnhDataMgmt   |
