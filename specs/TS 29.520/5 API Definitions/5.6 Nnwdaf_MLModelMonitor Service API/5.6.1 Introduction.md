# 5.6.1 Introduction


The Nnwdaf_MLModelMonitor service shall use the Nnwdaf_MLModelMonitor API.

The API URI of the Nnwdaf_MLModelMonitor API shall be:

{apiRoot}/\<apiName\>/\<apiVersion\>

The request URIs used in each HTTP requests from the NF service consumer towards the NWDAF shall have the Resource URI structure defined in clause 4.4.1 of 3GPP TS 29.501 \[7\], i.e.:

> **{apiRoot}/\<apiName\>/\<apiVersion\>/\<apiSpecificResourceUriPart\>**

with the following components:

\- The {apiRoot} shall be set as described in 3GPP TS 29.501 \[7\].

\- The\<apiName\> shall be "nnwdaf-mlmodelmonitor".

\- The \<apiVersion\> shall be "v1".

\- The \<apiSpecificResourceUriPart\> shall be set as described in clause 5.6.3.
