# 4.2.3.1 General

The Service Request procedure is used by a UE in CM-IDLE state or the 5GC to request the establishment of a secure connection to an AMF. The Service Request procedure is also used both when the UE is in CM-IDLE and in CM-CONNECTED to activate a User Plane connection for an established PDU Session. The Service Request procedure is also used to release the connection to an AMF.

For Home routed PDU sessions, by replacing the I-SMF with V-SMF and SMF with H-SMF the same procedure as defined in clause 4.23.4 is reused.

The UE shall not initiate a Service Request procedure if there is an ongoing Service Request procedure.
