# 1 Scope

The present document specifies the stage 3 protocol and data model for the MFAF Service Based Interface. It provides stage 3 protocol definitions and message flows, and specifies the API for each service offered by the MFAF.

The 5G System stage 2 architecture and procedures are specified in 3GPP TS 23.501 \[2\] and 3GPP TS 23.502 \[3\]. The stage 2 definition and procedures of Messaging Framework Adaptation are contained in 3GPP TS 23.288 \[14\] and 3GPP TS 23.502 \[3\].

The Technical Realization of the Service Based Architecture and the Principles and Guidelines for Services Definition are specified in 3GPP TS 29.500 \[4\] and 3GPP TS 29.501 \[5\].
