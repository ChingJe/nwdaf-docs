# 5.8 Data Types related to Charging


## 5.8.1 Introduction


This clause defines common data types related to Charging.

This clause specifies common simple data types.

<td>SmfChargingId</td>
<td>string</td>
<td><p>String based Charging ID as specified in TS 32.255 [58].</p>
<p>The String based Charging ID shall include a Uint32 base charging identifier (decimal encoded value within the values range: 0 to 4294967295 included) as the first segment, which shall be unique within the SMF assigning the Charging ID.</p>
<p>The String based Charging ID shall include the NF Instance ID (UUID format) of the SMF that assigned the Charging ID, as the second segment.</p>
<p>Pattern: '^(0|([1-9]{1}[0-9]{0,9}))\.smf-([0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12})$'</p>
<p>Example:</p>
<p>Base Charging ID:</p>
<blockquote>
<p>"9387"</p>
</blockquote>
<p>SMF NF Instance ID:</p>
<blockquote>
<p>"4947a69a-f61b-4bc1-b9da-47c9c5d14b64"</p>
</blockquote>
<p>String based Charging ID:</p>
<blockquote>
<p>"9387.smf-4947a69a-f61b-4bc1-b9da-47c9c5d14b64"</p>
</blockquote></td>
</tr>
