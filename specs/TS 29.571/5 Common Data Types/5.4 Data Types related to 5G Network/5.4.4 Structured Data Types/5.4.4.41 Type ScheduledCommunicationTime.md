# 5.4.4.41 Type: ScheduledCommunicationTime


Table 5.4.4.41-1: Definition of type ScheduledCommunicationTime

| Attribute name | Data type        | P   | Cardinality | Description                                                                       |
|----------------|------------------|-----|-------------|-----------------------------------------------------------------------------------|
