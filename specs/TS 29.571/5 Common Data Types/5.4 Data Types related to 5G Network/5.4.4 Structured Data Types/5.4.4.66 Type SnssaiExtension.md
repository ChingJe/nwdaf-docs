<tr class="odd">
<td colspan="5"><p>NOTE 1: sdRanges and wildcardSd shall not be present simultaneously.</p>
<p>NOTE 2: An SdRange may include the value “FFFFFF”; similarly, if wildcardSd is set to true, the SD value “FFFFFF” is one of the supported values. In both cases the SST without associated SD is one of the supported SNSSAIs, as the value “FFFFFF” indicates "no SD value associated with the SST" (see TS 23.003 [7]).</p></td>
