## 5.3.1 Introduction


This clause defines common data types related to subscription, identification and numbering information.

## 5.3.3 Enumerations


### 5.3.3.1 Enumeration: GroupServiceId


The enumeration GroupServiceId is a part of IMSI-Group-Id (see clause 19.9 of TS 23.003 \[7\]) and indicates the specific service for which the IMSI-Group-Id is used. Values greater than 1000 are reserved for home operator specific use. IMSI-Group-IDs with a Group-Service-Id in this range shall not be sent outside the HPLMN unless roaming agreements allow so.

Table 5.3.3.1-1: Enumeration GroupServiceId

| Enumeration value | Description                                                               |
|-------------------|---------------------------------------------------------------------------|
| 1                 | Group specific NAS level congestion control                               |
| 2                 | Group specific Monitoring of Number of UEs present in a geographical area |
| 3                 | Group specific for 5G LAN-type service                                    |
