# 5.6.2 Simple Data Types


This clause specifies common simple data types.

Table 5.6.2-1: Simple Data Types

<colgroup>
<col style="width: 20%" />
<col style="width: 20%" />

<td>Type Name</td>
<td>Type Definition</td>
<td>Description</td>

<tr class="even">
<td>PhysCellId</td>
<td>integer</td>
<td><p>integer value identifying the physical cell identity (PCI), as definition of "<em>PhysCellId</em>" IE in clause 6.3.2 of TS 38.331 [42].</p>
<p>Minimum = 0. Maximum = 1007.</p></td>
</tr>
<tr class="odd">
<td>ArfcnValueNR</td>
<td>integer</td>
<td><p>Integer value indicating the ARFCN applicable for a downlink, uplink or bi-directional (TDD) NR global frequency raster, as definition of "<em>ARFCN-ValueNR</em>" IE in clause 6.3.2 of TS 38.331 [42].</p>
<p>Minimum = 0. Maximum = 3279165.</p></td>
</tr>
<tr class="even">
<td>QoeReference</td>
<td>string</td>
<td><p>String containing MCC (3 digits), MNC (2 or 3 digits) and QMC ID (3 octets, encoded as 6 hexadecimal digits). Each value is separated by the "-" character.</p>
<p>See TS 28.405 [56], clause 5.2.</p>
<p>Pattern: '^[0-9]{3}-[0-9]{2,3}-[A-Fa-f0-9]{6}$'</p></td>
</tr>
<tr class="odd">
<td>MdtAlignmentInfo</td>
<td>string</td>
<td><p>String containing:</p>
<p>- Trace Reference: MCC (3 digits), MNC (2 or 3 digits), Trace ID (3 octets, encoded as 6 hexadecimal digits)</p>
<p>- Trace Recording Session Reference (2 octets, encoded as 4 hexadecimal digits).</p>
<p>Each value is separated by the "-" character.</p>
<p>See TS 28.405 [56], clause 5.13.</p>
<p>Pattern: '^[0-9]{3}-[0-9]{2,3}-[A-Fa-f0-9]{6}-[A-Fa-f0-9]{4}$'</p></td>
</tr>
