# 5.13.4 Used Features

The table below defines the features applicable to the NpConfiguration API. Those features are negotiated as described in clause 5.2.7.

<td>Notification_websocket</td>
<td>The delivery of notifications over Websocket is supported according to clause 5.2.5.4. This feature requires that the Notification_test_event featute is also supported.</td>

<td>Enhanced_param_config</td>
