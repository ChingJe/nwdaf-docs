# 5.14.2.1.16 Type: UeAddInfo

This type represents UEs Address information. It shall comply with the provisions defined in table 5.14.2.1.16-1.

Table 5.14.2.1.16-1: Definition of the type UeAddInfo

|                |           |             |                                                                                                                |               |
|----------------|-----------|-------------|----------------------------------------------------------------------------------------------------------------|---------------|
| Attribute name | Data type | Cardinality | Description                                                                                                    | Applicability |
| ueIpAddr       | IpAddr    | 1           | Identifies the UE IP address.                                                                                  |               |
| portNumber     | Port      | 0..1        | Indicates the UDP or TCP port number associated with the UE IP address as provided in the "ueIpAddr" attribute |               |
