# 4.1 Introduction

The Analytics Data Repository Service is used for the Analytics Data Repository Function (ADRF) to storage and retrieval of data and ML model(s) by e.g. NF service consumers (e.g. NWDAF) which access the data or ML model(s) using Nadrf service. The ADRF offers to NF service consumers the following services:
