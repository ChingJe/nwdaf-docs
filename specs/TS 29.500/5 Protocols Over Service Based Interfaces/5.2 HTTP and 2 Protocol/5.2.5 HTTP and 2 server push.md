# 5.2.5 HTTP/2 server push


HTTP/2 Server Push as specified in clause 8.4 of IETF RFC 9113 \[7\] may be supported and may be used by a NF Service Producer to proactively push resources to a NF Service Consumer, see clause 4.9.5 of 3GPP TS 29.501 \[5\].

A NF Service Consumer may choose to disable HTTP/2 Server Push by setting SETTINGS_ENABLE_PUSH to 0, as specified in clause 8.4 of IETF RFC 9113 \[7\].
