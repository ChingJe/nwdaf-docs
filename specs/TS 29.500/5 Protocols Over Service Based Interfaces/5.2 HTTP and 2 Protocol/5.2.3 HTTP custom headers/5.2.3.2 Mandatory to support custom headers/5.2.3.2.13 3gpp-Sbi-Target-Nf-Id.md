# 5.2.3.2.13 3gpp-Sbi-Target-Nf-Id


This header contains the target NF (Service) Instance ID in an HTTP 307/308 response (see clause 6.10.9).

The encoding of the header follows the ABNF as defined in IETF RFC 9110 \[11\].

Sbi-Target-Nf-Id-Header = "3gpp-Sbi-Target-Nf-Id:" OWS "nfinst=" nfinst \[ ";" OWS "nfservinst=" nfservinst \] OWS

The following parameters are defined:

\- nfinst (NF instance): indicates a NF Instance ID, as defined in 3GPP TS 29.510 \[8\]; the ABNF is defined in clause 5.2.3.2.8.

\- nfservinst (NF service instance): indicates a NF Service Instance ID, as defined in 3GPP TS 29.510 \[8\]; the ABNF is defined in clause 5.2.3.2.8.

EXAMPLE: 3gpp-Sbi-Target-Nf-Id: nfinst=54804518-4191-46b3-955c-ac631f953ed8; nfservinst=xyz
